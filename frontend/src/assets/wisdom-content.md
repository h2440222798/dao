## 一、核心理念

### ☯ 阴阳平衡 — 饮食之道

阴阳不是对立，而是互补。饮食中的寒热、温凉、补泻，都需要平衡。

**《黄帝内经》云：** "一阴一阳之谓道，偏阴偏阳之谓疾。"

汝且看——过食寒凉，阳气受损，脾胃不振；过食温热，阴液耗伤，内火上炎。真正的养生，在于"和"字。抗炎饮食追求的不是一味清热解毒，而是让身体回到阴阳平衡的状态。

**阴阳平衡饮食四要：**
- **寒热相济**：热性食物配凉性蔬菜，如羊肉炖白萝卜
- **补泻兼施**：滋补之中佐以通利，如人参配陈皮
- **升降相因**：升发之品配下降之物，如葱姜配蜂蜜
- **动静相宜**： animate 食物配静养之品，如动风之虾配静守之茯苓

### ☷ 五行调和 — 五脏之道

五行者，天地运行之规律也。木火土金水，相生相克，循环无端。

**《素问》曰：** "五行者，金木水火土也，更贵更贱，以知死生，以决成败。"

汝之五脏——肝心脾肺肾，正应五行。春养肝木，夏养心火，长夏养脾土，秋养肺金，冬养肾水。五色入五脏，五味调五气。

### 🌿 以食为养 — 食疗之道

**《黄帝内经》云：** "五谷为养，五果为助，五畜为益，五菜为充。"

上古之人，以食为药。药食同源，食疗为先。老夫常说：药补不如食补。一粥一饭，皆是养生之方；一蔬一果，俱是疗病之药。

## 二、养生格言十则

### 1. "饮食有节，起居有常，不妄作劳。" ——《素问》

**【现代解读】**
饮食要有节制，不能暴饮暴食；起居要有规律，不能昼夜颠倒；劳作要有度，不能过度透支。现代人最大的问题就是"无节"——想吃就吃、想熬就熬、想累就累，殊不知这正是慢性炎症的根源。

### 2. "五谷为养，五果为助，五畜为益，五菜为充。" ——《素问》

**【现代解读】**
五谷杂粮是养生的根本，水果蔬菜是辅助，肉类是补益，蔬菜是补充。现代人往往本末倒置——把肉当主食，把精制米面当主食，把蔬菜水果当配角。记住：粗粮才是养命之本。

### 3. "上医治未病，中医治欲病，下医治已病。" ——《黄帝内经》

**【现代解读】**
最高明的医生在你没病的时候就帮你预防，普通的医生在病初发时帮你调理，差劲的医生等到病重了才治病。抗炎饮食的精髓就在"治未病"——通过日常饮食调理，把炎症消灭在萌芽状态。

### 4. "春夏养阳，秋冬养阴。" ——《素问》

**【现代解读】**
春夏季节要养护阳气，多吃温热升发的食物；秋冬季节要养护阴液，多吃滋润潜藏的食物。顺应四时变化，身体才能与天地同步。

### 5. "形不足者，温之以气；精不足者，补之以味。" ——《素问》

**【现代解读】**
身体虚弱的人，要用温阳的方法来补；精气不足的人，要用美味的食物来养。抗炎不是一味清淡，该补的时候要补，该清的时候要清。

### 6. "胃不和则卧不安。" ——《素问》

**【现代解读】**
胃不舒服，睡觉就不安稳。晚上吃得过饱、过油腻，不仅影响睡眠，还会加重体内炎症。晚餐清淡、七分饱，是最好的安眠药。

### 7. "膏粱之变，足生大丁。" ——《素问》

**【现代解读】**
长期吃油腻精细的食物，容易生大病。现代的高油、高糖、高盐饮食，正是慢性炎症的温床。回归简单、天然的食物，身体自会感谢。

### 8. "法于阴阳，和于术数，食饮有节，起居有常。" ——《素问》

**【现代解读】**
效法阴阳变化的规律，配合养生的方法，饮食有节制，起居有规律。这是养生的总纲，做到这四点，百病不生。

### 9. "圣人不治已病治未病，不治已乱治未乱。" ——《黄帝内经》

**【现代解读】**
圣人不在病发后才治疗，而是在未病时就预防；不在大乱后才治理，而是在未乱时就防范。预防胜于治疗，日常饮食就是最好的预防 medicine。

### 10. "天食人以五气，地食人以五味。" ——《素问》

**【现代解读】**
天以五气（风暑湿燥寒）养人，地以五味（酸苦甘辛咸）养人。人活在天地之间，要顺应自然规律，吃应季的食物，才能得天地的滋养。

**【宜食】**
- **青色食物**：菠菜、芹菜、韭菜、西兰花、猕猴桃
- **酸味食物**：山楂、柠檬、醋、乌梅

**【忌食】**

## 四、阴阳平衡饮食原则

### ⚖️ 寒温平衡

**【原则】**
食物有寒热温凉之性，人体有阴阳偏盛之体。寒者热之，热者寒之，以平为期。

**【应用】**
- 阳虚怕冷者：多食生姜、羊肉、桂圆等温热之品
- 阴虚怕热者：多食绿豆、西瓜、苦瓜等寒凉之物
- 平性之人：寒热搭配，不寒不热

**【阴阳平衡口诀】**
```
寒热搭配要相宜，
过寒过热皆不宜。
阳虚怕冷食温热，
阴虚怕热食凉清。
```

### ⚖️ 补泻相宜

**【原则】**
虚则补之，实则泻之。补中有泻，泻中有补，方为正道。

**【应用】**
- 补气配行气：人参配陈皮
- 补血配活血：当归配川芎
- 补阴配利湿：熟地配茯苓

**【补泻平衡口诀】**
```
虚则补之实泻之，
补泻相宜是良医。
补中有泻不滞腻，
泻中有补不伤身。
```

### ⚖️ 动静结合

**【原则】**
食物有动静之分。动者升发，静者潜藏。动则生阳，静则生阴。

**【应用】**
- 动性食物：海鲜、禽类、葱姜蒜——升发阳气
- 静性食物：根茎类、豆类、坚果——涵养阴精
- 动静相配：虾配豆腐，鸡配山药

**【动静相宜口诀】**
```
动则升发静则藏，
动静相宜身体康。
升发太过需静守，
潜藏过久要动阳。
```

### ⚖️ 顺应天时

**【原则】**
春生夏长，秋收冬藏。顺应四时之气，调养五脏六腑。

**【应用】**
- 春季：食青升发，养肝疏泄
- 夏季：食赤清热，养心安神
- 长夏：食黄健脾，运化水湿
- 秋季：食白润燥，养肺收敛
- 冬季：食黑补肾，封藏精气

**【顺应天时口诀】**
```
春食青来夏食红，
秋食白来冬食黑。
长夏黄食养脾胃，
顺应四时寿百岁。
```

## 五、抗炎饮食的古法今用

### 🌿 抗炎即养阴

**【老祖宗怎么说】**
《黄帝内经》："阴虚则内热。"

**【现代解读】**
慢性炎症，在中医看来多为"阴虚内热"。抗炎的关键在于滋养阴液、清除虚热。深色蔬菜、优质油脂、全谷物，都是养阴的上品。

### 🌿 抗炎即健脾

**【老祖宗怎么说】**
《脾胃论》："百病皆由脾胃衰而生也。"

**【现代解读】**
肠道是免疫的大本营，脾胃是后天之本。抗炎要从健脾开始——少食生冷、细嚼慢咽、定时定量。脾胃健运，炎症自消。

### 🌿 抗炎即疏肝

**【老祖宗怎么说】**
《丹溪心法》："气血冲和，万病不生，一有怫郁，诸病生焉。"

**【现代解读】**
情绪与炎症密切相关。压力、焦虑、抑郁都会导致体内炎症因子升高。疏肝理气，保持心情愉悦，是最好的抗炎药。
